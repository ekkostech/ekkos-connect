These logos are used in the ekkOS VS Code extension to:
1. Display IDE-specific branding in documentation and UI
2. Show OS-specific instructions and compatibility
3. Indicate service integrations and dependencies

## Updates

Last updated: 2026-01-12

To update logos:
```bash
# Update from jsdelivr CDN
curl -s "https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/<icon-name>.svg" -o <path>

# Or from simpleicons.org CDN
curl -s "https://cdn.simpleicons.org/<icon-name>" -o <path>
```
