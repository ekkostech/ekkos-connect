## Usage in ekkOS Extension

These logos are used throughout the ekkOS VS Code extension to:
- Display AI provider icons in settings
- Show active model provider in status bar
- Identify different LLM services in the UI
- Enhance visual recognition of AI providers

## File Sizes

All logos are optimized SVG files, typically under 5KB each:
- Total size: ~12KB for all 11 logos
- Format: SVG (Scalable Vector Graphics)
- Compatible with: Web, VS Code extensions, React, Vue

## Brand Guidelines

When using these logos:
- Maintain aspect ratio

- Follow each provider's brand guidelines
- Ensure proper attribution per license requirements

## Notes

- **Gemini** and **Google** are separate - use Gemini for the specific model, Google for general Google AI
- **xAI** uses the X logo - this is the official branding for Grok
- **Meta** represents Meta AI and Llama models
- **DeepMind** is technically part of Google but maintains separate branding
